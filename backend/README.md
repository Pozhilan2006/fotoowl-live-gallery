# Backend

Minimal Express backend for FotoOwl Live Gallery.

Quick start:

1. Copy `.env.example` to `.env` and adjust values.
2. Install dependencies: `npm install` (run in the `backend` folder).
3. Start in development: `npm run dev` (requires `nodemon`).

Endpoints:

- `GET /` - health check
- `GET /images` - placeholder images list
